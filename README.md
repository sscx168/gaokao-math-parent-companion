# gaokao-math-parent-companion

高考数学家长伴学工具。一个单文件 HTML 应用，帮不懂数学的家长也能参与孩子的高考备考。

## 背景

高三家长想帮孩子，但大多数人数学早就忘了。看不懂题，不知道说什么，想鼓励又怕说错。这个工具解决的就是这个问题：不需要你会做题，只需要你知道什么时候说什么话。

## 包含什么

**备考全局** — 一张图看清高考数学考什么、怎么考、各板块的提分难度。给家长建立基本认知，不用深入细节。

**30天特训** — 每天花10分钟读一个数学概念。每个概念都有三个版本：一句话版本（先抓住核心）、成人映射版（用商业/生活场景类比）、亲子话术模板（直接能跟孩子说的话）。

**真题拆解** — 七大板块（函数、三角、数列、立体几何、解析几何、导数、概率）的高考大题 SOP 步骤演示。点一下看一步，像拆机器一样看懂一道题是怎么被标准流程解决的。

**伴读工具箱** — 这是给家长用的核心模块：
- 速查手册：用一句话和一个比方解释每个板块，附带跟孩子聊天的示范对话
- 对话剧本：五个高频场景（开始学习前、卡住了、考砸了、不想学了、有进步时），每个场景三句话，拿来就能说
- 情绪急救：孩子焦虑、暴躁、哭了的时候怎么办，给具体的应对话术
- 错题诊断：跟着选项点，三步定位孩子卡在哪里，给出针对性建议
- 每日任务：每天一个5分钟以内的亲子互动小任务
- 伴读成长：五个等级，从旁观者到搭档，记录你的进步

**AI辅助** — 接入任何兼容 OpenAI 接口的大模型（OpenAI、DeepSeek、通义千问、Kimi 等），提供两个功能：难题概念脱水（把抽象数学变成商战案例）、亲子共情信生成（用数学隐喻写一封鼓励信）。

**七大板块交互演示** — 每个板块都有一个可操作的可视化 demo，拖动滑块看函数图像变化、看三角函数波形、看高尔顿板小球分布等。

## 怎么用

不需要安装任何东西。下载 HTML 文件，用浏览器打开即可。

```
直接双击 code_artifact.html
```

AI 功能需要配置 API 地址和密钥，在页面里的 AI 辅助板块填写。

## 技术说明

单文件 HTML，不依赖后端。用到的外部资源：
- Tailwind CSS（CDN）
- MathJax（数学公式渲染，CDN）
- Font Awesome（图标，CDN）
- Chart.js（图表，CDN）

数据保存在浏览器 localStorage，包括学习进度、AI 配置、伴读等级。

## 适合谁

- 高三学生家长，数学基础薄弱或已经遗忘
- 想参与孩子备考但不知道怎么开口的家长
- 需要具体话术而不是抽象建议的家长

## 不适合谁

- 需要系统性数学教学的用户（这是伴学工具，不是教学软件）
- 需要在线刷题或模拟考试的用户
