1. 什么是函数？

函数・高频高考真题3道脱水解刨

母题1：送分题・奇偶数求对称点

【真题形式】已知 $f(x)$ 是奇函数，在 $x>0$ 时表达式为 $f(x)=2x+1$，求在 $x<0$ 时的公式。

【家长极简拆解】 奇函数的意思就是图形绕着中心180度旋转能完全重合。因为是对称的，孩子只需要把原公式里所有的 $x$ 改写成 $-x$，并在最外面加个负号。即 $f(-x) = -f(x)$。这就好比公司的左右镜面对称业务，计算量极低。

母题2：中档题・零点区间逼近

【真题形式】判断复杂方程 $x^3 - 3x - 1 = 0$ 在哪个区间内有实数解？

【家长极简拆解】 这叫零点存在性定理。家长只要告诉孩子：如果函数在自变量等于 $1$ 时输出是负数 $f(1) < 0$，自变量等于 $2$ 时输出是正数 $f(2) > 0$。那么在 $1$ 和 $2$ 之间，由于曲线是连续的，它必定穿过了 $x$ 轴（即穿过了 $y=0$ 的零点线）。这叫过河理论。

母题3：压轴题・构造函数比大小

【真题形式】已知 $a = \ln 1.1$, $b = 0.1$, $c = e^{0.1} - 1$，比较 $a,b,c$ 的大小。

【家长极简拆解】 这类题硬算是不可能算出来的。核心策略是：把数字改成自变量 $x$（设 $x=0.1$），把三个选项改写成三个基本函数 $y=\ln(1+x)$，$y=x$ 和 $y=e^x-1$。然后让孩子求导画出这三个走势图，利用图像在 $x \in (0,1)$ 的高度一目了然比出 $a < b < c$ 的关系。

🌉 顶层策略桥梁：从定常角比到周期律波动圆盘

经典常态认知：以前我们局限于直角三角形里的比值关系：正弦 $\sin A = \frac{\text{对边}}{\text{斜边}}$。因为三角形角度必须是锐角，所以是静止静默的。
高考战术跃迁：高考将这根斜边拉到直角坐标系中，设长度为 $1$，让它疯狂转动！这被称为单位圆模型。随着端点转过 $360^\circ$ 甚至是数个周期，它的投影在纵轴（正弦）和横轴（余弦）上不断重演，呈现出完美的波动波浪曲线。高考试图用三角函数公式（如辅助角公式、诱导公式）来精确调谐那些关于行业大震荡、发电机自感电动势、以及一切包含周期起伏循环的复杂系统。

三角函数・高频高考真题3道脱水解刨

母题1：送分题・图像位移缩放

【真题形式】如何将 $y=\sin x$ 变换得到 $y=2\sin(2x + \frac{\pi}{3})$？

【家长极简拆解】 这相当于给一个振荡周期产品进行频率翻倍（周期由 $2\pi$ 变为 $\pi$），振幅翻倍（最大值变为 $2$），并整体提前运行（初相向左移动 $\frac{\pi}{6}$）。记住平移量一定是提公因数后的值，即由 $2(x+\frac{\pi}{6})$ 决定平移 $\frac{\pi}{6}$。

母题2：中档题・正弦余弦定理与三角形

【真题形式】在 $\triangle ABC$ 中，已知 $a\sin B = \sqrt{3}b\cos A$，求角 $A$。

【家长极简拆解】 这是最标准的边角互化策略。根据正弦定理，边长之比就是对应的正弦之比 $\frac{a}{\sin A} = \frac{b}{\sin B}$。直接将上式中的 $a$ 换成 $\sin A$，$b$ 换成 $\sin B$。消元后得到 $\sin A\sin B = \sqrt{3}\sin B\cos A$。由于角 $B$ 是三角形内角，正弦值一定不为0，两边同除以 $\sin B$，立刻得到 $\tan A = \sqrt{3}$，解出角 $A = \frac{\pi}{3}$ ($60^\circ$)。

母题3：压轴题・辅助角公式统一叠加

【真题形式】求复杂函数 $y = \sqrt{3}\sin x + \cos x$ 的振幅与最高值点。

【家长极简拆解】 两个不同波谷和周期的波叠加，在数学家眼里其实可以通过辅助角公式合并为一个波：$y = A\sin(x+\phi)$，其中合体振幅 $A = \sqrt{(\sqrt{3})^2 + 1^2} = 2$。合体为 $y=2\sin(x+\frac{\pi}{6})$。所以该波最大顶峰高度为 $2$。

🌉 顶层策略桥梁：从散乱规律到确定性财富增长算盘

经典常态认知：以前我们凭借脑力去拼填空找规律：如 $1, 4, 7, 10, \dots$ 的下一个数字是什么。
高考战术跃迁：高考将离散的序列编成具有严格编号的元素 $a_n$。主要构建了两种完美增长架构：**等差数列**（公差为 $d$，稳定的线性单利增加，像稳健年薪工资） and **等比数列**（公比为 $q$，几何级的爆发式复利相乘，像资本市场滚雪球）。高考研究怎样利用相邻元素的关系式（递推公式 $a_{n+1} = f(a_n)$）去锁定任意天数下的通项值，并巧设错位相减或裂项相消机制把一整张复杂的资产流水直接抵消打包清算（求和 $S_n$）。

成人底层逻辑映射：时间复利之美

1. 等差数列（打工模式）：
每天给你固定50块（$a_n = a_1 + (n-1)d$）。这就是单利模式，在早期它看起来很棒，但随着时间推移，其天花板极低。

2. 等比数列（投资裂变）：
每天在昨天的基础上翻倍（$a_n = a_1 \cdot q^{n-1}$）。在初期，等比数列可能跑得比等差慢很多，一旦跨越临界点（拐点），就能爆发出超越想象的力量。

数列・高频高考真题3道脱水解刨

母题1：送分题・方程法求基本量

【真题形式】在等差数列 $\{a_n\}$ 中，已知 $a_3 = 5$, $a_7 = 13$，求其通项公式。

【家长极简拆解】 这图像资产盘点：第3天资产是5，第7天资产是13。相差的4天里资产平稳增长了8。所以公差 $d = \frac{13-5}{7-3} = 2$。倒推第一天首项 $a_1 = a_3 - 2d = 1$。所以通项公式 $a_n = 1 + (n-1)\cdot2 = 2n-1$。全套逻辑都是极其直白的小学方程式。

母题2：中档题・错位相减法求和

【真题形式】求数列 $S_n = 1\cdot2^1 + 2\cdot2^2 + 3\cdot2^3 + \dots + n\cdot2^n$ 的和。

【家长极简拆解】 这是一个由等差 $1,2,3$和等比 $2^1,2^2,2^3$相乘揉合而成的复合账目。只需让整条等式两边同时乘以等比公比 $2$。此时所有项在右边错开一位。用旧等式减去新等式（$S_n - 2S_n$），中间除了首尾外，所有的复合复杂项会自动相扣、消成纯等比数列求和。这在高考中值足足 6 分步骤分！

母题3：压轴题・裂项相消求和

【真题形式】求和：$S_n = \frac{1}{1\cdot2} + \frac{1}{2\cdot3} + \dots + \frac{1}{n(n+1)}$。

【家长极简拆解】 将每一项拆解开：$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$。所以整条式子拉开后成为：$(1-\frac{1}{2}) + (\frac{1}{2}-\frac{1}{3}) + \dots + (\frac{1}{n}-\frac{1}{n+1})$。中间一正一负形成多米诺骨牌相扣连锁抵消，最后只剩首尾相减得到 $1 - \frac{1}{n+1}$。这在数学上叫裂开相消，简化大盘结算。

解析几何・高频高考真题3道脱水解刨

母题1：送分题・离心率快速通关

【真题形式】求椭圆 $\frac{x^2}{4} + \frac{y^2}{3} = 1$ 的离心率 $e$。

【家长极简拆解】 由分母可知，横向半长平方 $a^2=4$（即 $a=2$），纵向半长平方 $b^2=3$。根据椭圆特征参数公式 $a^2 = b^2 + c^2$，解出焦点半距平方 $c^2 = a^2 - b^2 = 4-3 = 1$（即 $c=1$）。所以离心率为：$e = \frac{c}{a} = \frac{1}{2} = 0.5$。这题白给！

母题2：中档拉分·中点弦与点差法

【真题形式】已知直线与椭圆 $\frac{x^2}{4} + y^2 = 1$ 相交，中点为 $M(1,\frac{1}{2})$，求直线斜率 $k$。

【家长极简拆解】 将两端点坐标代入椭圆式子相减。通过平方差公式化简，可以直接得出斜率 $k$、中点坐标 $(x_0, y_0)$ 之间的瞬时速算比例：$k = -\frac{b^2 x_0}{a^2 y_0}$。代入数字得 $k = -\frac{1\cdot1}{4 \cdot 1/2} = -0.5$。直接秒杀！

母题3：压轴难题·直线与椭圆大相切/相交

【真题形式】直线与椭圆联立，求动弦长度 $AB$ 或夹角的最值。

【家长极简拆解】 高考最大代数屠宰场。不要怕，它有雷打不动的标准流程：①设直线 $y=kx+m$。②代入椭圆，化成一元二次方程。③套用韦达定理写出 $x_1+x_2$ 和 $x_1x_2$。④代入长度弦长公式 $d = \sqrt{1+k^2}|x_1-x_2|$ 进行纯计算。只要孩子心平气和做完，12分拿个10分板上钉钉！

母题1：送分题・切线斜率代入法

【真题形式】已知函数 $f(x) = x^2 + \ln x$，求其在点 $(1,1)$ 处的切线。

【家长极简拆解】 ①求导得出切斜手电筒公式 $f'(x) = 2x + \frac{1}{x}$。②代入交点横坐标 $x=1$，得出切线的斜率 $k = 2(1) + \frac{1}{1} = 3$。③代入直线斜截式得到切线方程：$y-1 = 3(x-1)$。纯粹的加减法求点，毫无技术难度。

母题2：中档拉分·求极大值与极小值

【真题形式】求复杂函数 $f(x) = x e^{-x}$ 的单调区间 and 极大值点。

【家长极简拆解】 寻曲线最高点：第一步求导：$f'(x) = (1-x)e^{-x}$。第二步令坡度等于0 $f'(x)=0$，解出山峰交接点 $x=1$。第三步画出单调表（$x<1$ 时爬坡递增，$x>1$ 时下行递减），所以极大值必在顶点 $x=1$ 处取得，数值为 $1\cdot e^{-1} = \frac{1}{e}$。这就是标准的爬坡图绘表。

母题3：压轴难题·超越方程与不等式恒成立

【真题形式】已知对于一切 $x>0$ 都有 $e^x - a x - 1 \ge 0$ 恒成立，求 $a$ 范围。

【家长极简拆解】 终极防御问题。只需使用差值构造法：设新函数 $F(x) = e^x - ax - 1$。为了保证新函数对于所有正数输出都大于等于0，只要保证它的最低谷（极小值）不跌破零点即可。对 $F(x)$求导找极小值点并令其大于零，即可锁定参数限制 $a \le 1$。这叫守住低回撤底线理论。
